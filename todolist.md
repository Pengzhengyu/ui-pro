一、 极致交互类 (User Experience)
Command Palette (全局指令面板)

唤起与隐藏：支持全局快捷键（如 Cmd+K）唤起，点击遮罩或 Esc 关闭。

搜索过滤：输入文字实时过滤指令列表，支持拼音模糊匹配和关键词高亮。

分组展示：支持将结果分为“常用”、“导航”、“动作”等类别。

最近访问：自动记录用户最近执行过的 5 条指令并置顶。

二级联动：点击某个指令（如“切换语言”）进入子菜单，支持返回上一级。

Spring Draggable List (弹性物理拖拽列表)

实时排序：拖拽 A 元素经过 B 元素时，B 自动平滑移位腾出空间。

物理反馈：拖动时组件随速度产生轻微的拉伸、倾斜变形动画。

吸附效果：松开鼠标后，组件自动“弹动”回到最终位置，而非生硬停止。

跨组拖拽：支持将元素从 A 列表拖入 B 列表，并伴随容器高亮反馈。

Sticky Section Scroller (视差粘性容器)

左图右文联动：左侧图片随滚动条固定，右侧文字滚动；文字切换到新章节时，左侧图片自动切换。

进度指示：侧边栏实时显示当前阅读进度（长条或百分比）。

锚点跳转：点击侧边标题，内容区平滑滚动到指定位置。

Floating Dock (仿 Mac 浮动工具栏)

鱼眼缩放：鼠标靠近图标时，图标及邻近图标产生平滑的放大曲线。

气泡提示：鼠标悬停显示工具名称小气泡。

自动避让：当页面弹窗或其他元素重叠时，Dock 自动降低透明度或缩小。

二、 开发者与效率类 (Efficiency Tools)
JSON Schema Visual Editor (可视化配置器)

树状编辑：通过点击 + 号增加对象、数组或基础字段。

类型切换：点击下拉框切换字段类型（String, Number, Boolean, Object）。

实时校验：输入内容不符合类型定义时，红框提示错误信息。

导入导出：支持直接粘贴一段 JSON 生成 UI，或点击复制当前配置生成的 JSON。

Code Playground (组件沙盒预览)

实时渲染：左侧编辑代码（如 React/Vue 代码块），右侧实时渲染出 UI。

环境预设：支持一键切换不同的背景色（黑/白/透明）、容器尺寸（手机/平板/电脑）。

控制台日志：内置微型 Console，展示代码运行中的 console.log 信息。

Keybinding Monitor (快捷键录制/显示)

组合键捕获：用户按下键盘时，UI 实时显示出 Ctrl + Shift + P 这样的美化图标。

冲突检查：录制时自动检测是否与系统或已有业务快捷键冲突。

清单生成：自动根据传入的 Actions 列表生成一份精美的快捷键说明书。

Network Mock Manager (接口模拟悬浮窗)

状态切换：一键开启/关闭 Mock 拦截。

延时模拟：通过滑块设置接口响应延迟时间（如 500ms、3000ms）。

模拟报错：强行设置接口返回 404、500 等状态码进行边界测试。

三、 类 Excel 与复杂数据类 (Data Intensive)
Canvas Area Selector (画布区域选择器)

框选逻辑：按住左键拖动产生蓝色半透明选框。

多选反馈：框选覆盖到的元素自动进入“选中态”（如加边框）。

增量选择：按住 Shift 划选可增加选中项，按住 Ctrl 划选可剔除已选中项。

Custom Context Menu (智能右键菜单)

层级结构：支持无限级子菜单嵌套。

动态项：根据右键点击的对象不同，展示不同的菜单项（如点图片显示“保存”，点文字显示“复制”）。

快捷键映射：菜单项右侧展示对应快捷键，且按下对应键可直接触发菜单动作。

Spreadsheet Editor (轻量电子表格)

双击编辑：双击单元格弹出输入框，按 Enter 确认，按 Esc 取消并还原。

行列操作：右键表头可进行“左侧插入列”、“删除当前行”、“隐藏列”操作。

单元格合并：选中多个区域后，支持合并为一个大单元格。

内容格式化：支持设置单元格为货币、百分比、日期等展示格式。

Gantt Timeline (任务甘特图)

任务拖拽：在时间轴上左右拖拽改变任务起止时间。

依赖连线：点击任务末端并拖线到另一个任务，建立“前置/后置”关系。

进度填充：任务块内部显示深色进度条，代表当前完成百分比。

Pivot Table (数据透视组件)

维度切换：支持将字段从“备选区”拖入“行区”、“列区”或“值区”。

自动汇总：根据拖入的字段，自动计算总计、平均值、计数等。

四、 垂直业务场景 (Business Logic)
Time-Travel Comparison (时间轴对比)

版本切换：点击时间轴上的不同点位，左右分栏展示两个时间点的配置差异。

增量高亮：绿色代表新增，红色代表删除，黄色代表修改。

Org Chart Editor (组织架构编辑器)

节点拖放：支持将一个部门或人拖到另一个部门下方，自动重新计算层级。

快速收起：点击节点上的 - 号隐藏其所有下属节点。

Dynamic Watermark (防御水印)

防删检测：一旦用户从 F12 删除了水印节点，组件立即重新渲染出新水印。

动态赋值：水印自动获取当前用户的工号、登录 IP 和操作时间。

Step Form Persistence (带记忆的分步表单)

断点续填：用户填到第三步关闭浏览器，再次打开时自动恢复前两步已填内容。

逻辑路由：根据第一步的选择（如：个人/企业），动态决定第二步展示的表单项。

Layout Splitter (可调布局容器)

自由拉伸：拖动两面板间的细线改变宽度或高度比例。

折叠按钮：分割线上带有一个微型箭头，点击可一键隐藏左侧或右侧面板。

Smart Token Input (智能标签输入框)

自动补全：输入 @ 弹出用户列表，输入 # 弹出标签列表。

拖拽排序：生成的标签（Token）支持在输入框内自由拖动调整顺序。

Interactive Tour (新手引导)

高亮聚焦：页面变暗，仅高亮显示目标组件。

强制交互：支持设置“必须点击此按钮才能进行下一步”，而非简单的展示。
