# Antigravity UI Pro

一套基于 **React (Hooks)** 和 **Webpack** 构建的高性能、未来感组件库。本项目从原始的 Vanilla JS 版本全面进化，注入了虚拟滚动技术与富有创意的交互体验。

## ✨ 特色组件 (Premium Bundle)

### 1. Virtualized Intelligence (Tree Table)
- **高性能虚拟滚动**：丝滑处理 10,000+ 级大数据。
- **树形逻辑**：深度集成递归展开与折叠。
- **Excel 级选区**：支持单元格/行的跨区域拖拽选区，并内置 Excel/TSV 粘贴解析器。
- **响应式列宽**：已修复拖拽逻辑，支持横向滚动同步。

### 2. Liquid Interaction (Mercury Button)
- **流体融合反馈**：利用 SVG 滤镜模拟水银般的液态交互，赋予操作极致的生命感。

### 3. Holographic Card (3D Reflection)
- **全息光栅化**：卡片随鼠标移动产生 3D 旋转与动态光影折射。

### 4. Cyber Scanner (HUD Wrapper)
- **赛博扫描器**：为任意内容增加高科技扫描线、HUD 边角与层级脉冲动画。

### 5. Neural Network (Dynamic Canvas)
- **实时背景系统**：Canvas 渲染的动态节点网络，营造极致的科技氛围。

## 🛠 技术栈
- **Core**: React 18 (Hooks)
- **Build**: Webpack 5 + Babel 7
- **Styling**: Pure CSS (Modular patterns)
- **Aesthetics**: Cyberpunk, Glassmorphism, Premium Dark Theme.

## 🚀 快速启动

1. **安装依赖**:
   ```bash
   npm install
   ```

2. **开发模式 (Webpack Dev Server)**:
   ```bash
   npm start
   ```
   访问 `http://localhost:3000` 查看高级演示面板。

3. **生产构建**:
   ```bash
   npm run build
   ```
